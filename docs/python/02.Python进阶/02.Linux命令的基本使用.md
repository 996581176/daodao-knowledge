## Linux命令
Linux操作系统在服务器方面应用很广泛，而在操作服务器相关设备的时候往往是没有图形化界面的，这里就需要我们使用Linux命令来进行操作。

## 查看目录命令
|命令|说明|
|---|----|
|ls |查看当前路径下的目录信息|
|tree|以树状方式显示目录|
|pwd |查看当前目录路径|
|clear|清除终端内容|
