## 地图演示
```py
"""
演示地图可视化的基本使用
"""
from pyecharts.charts import Map

# 准备地图对象
map = Map()

# 准备数据
data = [
    ("北京市",99),
    ("上海市", 199),
    ("湖南省", 299),
    ("台湾省", 399),
    ("广东省", 499),
]


# 添加数据
map.add("测试地图",data,"china")

# 绘图
map.render()
```
